## Présentation du Projet

Ce projet est une expérience web interactive créée pour la Nuit de l'Info. Il met en scène une comparaison satirique entre deux univers informatiques : **Windows XP** (représentant le monde propriétaire avec ses travers) et **Ubuntu/Linux** (symbolisant le monde libre et open source).

Le projet vise à **sensibiliser** les utilisateurs sur plusieurs thématiques importantes :  
\- La protection des données et la vie privée  
\- Les dangers de la surveillance numérique  
\- L'importance des logiciels libres et open source  
\- La place des femmes dans l'informatique  
\- La sécurité numérique et les bonnes pratiques

## Structure et URLs du Projet

* Page d'Accueil  
  * URL principale : /index.html  
  * Redirige automatiquement vers l'univers Windows XP (/windows/windows.html)  
* Univers Windows XP \- "Le Monde Propriétaire" :  
  * Page Principale Windows  
    * URL : /windows/windows.html  
    * Description : Bureau Windows XP satirique avec des pop-ups, publicités et références aux problèmes du monde propriétaire  
    * Fonctionnalités :  
      * Interface Windows XP authentique  
      * Barre des tâches interactive  
      * Menu démarrer fonctionnel  
      * Horloge en temps réel  
      * Icônes du bureau cliquables  
      * Bouton "SAUVER LE PC" pour passer vers Linux  
      * Systeme ergonomic non standard dans les popup (L'ergonomie : simplifier pour mieux vivre)  
  * Applications Windows : Les icônes du bureau donnent accès à différentes fonctionnalités  
    * 1\. Internet Explore  
      * Application fictive générant des pop-ups publicitaires  
      * Simule les problèmes de navigation du passé  
    * 2\. CatGPNulle (Chat’Bruti)  
      * Chatbot parodique   
      * Simule un assistant peu utile  
    * 3\. Windows Nous Surveille  
      * URL : /windows/apps/surveillance.html  
      * Accecile aussi directement dans l’interface windows (/windows/windows.html)  
      * Description : Page de sensibilisation interactive sur la surveillance et la confidentialité  
    * 4\. Rechercher (Chien Roger)  
      * Référence nostalgique à l'assistant de recherche de Windows XP  
      * Application qui avait eu une faille de sécurité faisant que l’on pouvait exécuté des commandes systeme via la barre de recherche (La faille la plus maligne)  
    * 5\. Mon Ordinateur  
      * Affiche des popup système parodiques  
    * 6\. Corbeill  
      * Élément décoratif du bureau  
      * Affiche une popup parodique  
  * Fonctionnalité Spéciale : BSOD (Blue Screen of Death)  
    * Écran bleu satirique accessible via le menu démarrer  
    * Message humoristique invitant à passer vers Linux  
    * Transition interactive vers l'univers Ubuntu  
* Univers Ubuntu/Linux \- "Le Monde Libre"  
  * Page Principale Ubuntu  
    * URL : /ubuntu/ubuntu.html  
    * Description : Bureau Ubuntu moderne et épuré  
    * Éléments d'interface :  
      * Barre supérieure avec informations système et horloge  
      * Sidebar avec icônes d'applications  
      * Bureau avec icônes organisées en grille  
      * Fond d'écran Ubuntu authentique  
  * Applications Ubuntu  
    * 1\. Le Monde Libre  
      * URL : /ubuntu/apps/demarche-libre.html  
      * Description : Article éducatif sur la démarche libre et open source  
      * Contenu :  
        * Définition de l'informatique libre  
        * Avantages du logiciel libre (transparence, autonomie, sécurité)  
        * Impact dans différents domaines (éducation, science, administration)  
        * Exemples concrets de logiciels libres  
        * Ressources pour aller plus loin  
        * Snake caché dans le coin en bas a droite (Hidden Snake)  
    * 2\. Femmes et Informatique  
      * URL : /ubuntu/apps/femmes-informatique.html  
      * Description : Page de sensibilisation sur la place des femmes dans l'informatique  
      * Contenu :  
        * Histoire des pionnières (Ada Lovelace, etc.)  
        * État actuel de la mixité dans le numérique  
        * Vidéo intégrée sur le sujet  
        * Flyer téléchargeable (PDF)  
        * Sensibilisation à l'importance de la diversité  
    * 3\. Quiz  
      * URL : /ubuntu/apps/quiz.html  
      * Description : Quiz interactif sur le numérique, la sécurité et la culture informatique  
      * Fonctionnalités :  
        * Questions à choix multiples  
        * Feedback immédiat sur les réponses  
        * Navigation entre questions  
    * 4\. Terminal  
      * URL : /ubuntu/apps/terminal.html  
      * Description : Terminal Ubuntu qui sert au crédit du projet  
    * 5\. Paramètres  
      * URL : /ubuntu/apps/parametres.html  
      * Description : Panneau de paramètres système (si implémenté)  
    * 6\. Retour en Enfer  
      * Icône spéciale permettant de retourner vers l'univers Windows XP
